# rebase & squash


#### 현재 커밋 기록 먼저 확인
```bash
git log --all --decorate --graph --oneline

```
#### 합칠 commit들 선택 후 rebase
```bash
git rebase -i HEAD~4
```
합치고 싶은 커밋 개수 (Head 로부터 몇개 없앨건지)
=> 2개면 Head 와 그 전 커밋 합침
