## Question
- 다이아몬드 별찍기 
```
    *
   ***
  *****
 *******
*********
```



## Answer

> [!tip]
> 실수 말고 한번 더 생각하기

``` java
for (int i = 1; i <= cnt; i++) {
            String empty = " ".repeat(cnt - i);
            String star = "*".repeat(2 * i - 1);
            System.out.println(empty + star);
        }
```
