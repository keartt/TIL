# 이분탐색

## 이분탐색이란?
정렬된 배열에서 특정 값을 찾기 위한 알고리즘 중 하나로,
배열에서 중간값을 기준으로 데이터를 탐색한다, 
> 데이터가 정렬되어 있어야 함

## 시간복잡도
$O(logN)$
