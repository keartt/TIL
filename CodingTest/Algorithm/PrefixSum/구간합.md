## 합배열 S

`S[i] = A[0] + a[1] + ... + A[i-1] + A[i]` <br>
기존의 배열을 전처리한 배열 <br>
기존 일정 범위의 합을 구하는 시간복잡도가 $O(N)$ 에서 $O(1)$ 로 감소 <br>

### 합배열 만드는 법
```java
S[i] = S[i-1] + A[i];
```
